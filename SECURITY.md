## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please visit [https://sec.chkdsklabs.io](https://sec.chkdsklabs.io/) and submit a support request with the following information:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

In the event that the security site is unavailable, please email **jay@chkdsklabs.com** with the above information.


You can expect a response within **72 hours**. We'll work with you to
understand and address the issue before any public disclosure.

Our full disclosure policy is available at [https://sec.chkdsklabs.io/disclosure](https://sec.chkdsklabs.io/disclosure). We follow responsible disclosure practices and will credit you for your contribution if you choose to be named.
